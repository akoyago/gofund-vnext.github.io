GOfund vNext — Candidate Features

Target market: Community Foundations Date: 2026-03-28 Ground-up rewrite — includes existing features (carry forward) and new features (gaps identified from competitive research) Sources: GOfund Feature Catalog, Feature Gap Analysis, Similar Products Research

How to Read This Document

This is a complete feature list for the GOfund vNext rewrite. Every feature is tagged:

  • EXISTING = Carry forward from current GOfund (already built, needs rewrite for new architecture)
  • NEW = Gap feature identified from competitive research (not in current GOfund)
  • ENHANCE = Exists in current GOfund but needs significant improvement based on competitive gaps

Priority tiers:

  • P0 — Must Have: Ship without these and you can't sell the product
  • P1 — Should Have: Close the gap with Foundant CommunitySuite, Give Interactive, and Fidelity Charitable
  • P2 — Differentiators: Whitespace features where GOfund can leapfrog competitors
  • P3 — Future: Defer unless customer demand emerges

Market tags:

  • CF = Community Foundation priority
  • DAF = Primarily DAF feature (defer for CF launch)
  • CF+DAF = Applies to both

P1 — Should Have

Grant Recommendation Enhancements

# Feature Status Type Description
F-58 Rich grant use taxonomy NEW CF+DAF Structured "Use" dropdown: Where it's needed most, Annual fund, In honor of (with note), In memory of (with note), Membership, Pledge, Tithe, Create your own. CF fundholders frequently make grants "in memory of" or "in honor of." Fidelity offers 12 use types.

P2 — Differentiators

These are features where GOfund can leapfrog competitors. No one owns these in the donor portal today.

# Feature Status Type Description
F-69 Geographic grant distribution map NEW CF Interactive map showing where the donor's grants went — pins with org names and amounts. No competitor has this in a donor portal. Especially valuable for CF fundholders whose grants focus on a local community.
F-70 AI-powered organization discovery NEW CF+DAF Donor describes philanthropic interests in plain language → gets AI-recommended organizations with relevance explanations. No competitor offers this. Especially valuable for CFs: "find local nonprofits in our community that match my interests."
F-71 AI grant impact summaries NEW CF+DAF AI-generated plain-language summary of what the donor's past grants achieved — synthesized from grantee reports (via GOapply). "Your $10K grant to Habitat for Humanity helped build 3 homes in your community." Cross-product differentiator.

Implementation Notes

Quick Wins (low complexity, high visibility — do first)

  • F-28 Grant status tracking — expose existing CRM status field to the portal
  • F-59 Grant acknowledgment options — add dropdown to existing grant form
  • F-58 Rich grant use taxonomy — add dropdown to existing grant form
  • F-61 Contribution history print/download — add toolbar buttons to existing view
  • F-29 Scheduled grant management — list view over existing scheduled grant data

Data Dependencies (need CRM-side work first)

  • F-33 Tax receipts require integration with contribution records and PDF generation
  • F-37 Investment performance requires time-series data from Business Central or custodial integration
  • F-36 Grant analysis requires grants to be tagged with cause area/sector codes (CRM-side tagging)
  • F-71 AI impact summaries require grantee reports from GOapply connected to donor grant records
  • F-39 Geographic map requires geo-coded organization addresses

UX-Heavy (design investment needed)

  • F-11 Fund dashboard homepage — defines the first impression; needs careful design
  • F-36 Grant analysis dashboard — data visualization UX
  • F-39 Geographic grant map — interactive map component
  • F-42 Grant catalog — browsable marketplace UX with foundation staff curation workflow
  • F-70 AI org discovery — NLP search interface with AI infrastructure

CF-Specific Features to Highlight in Sales

These differentiate GOfund for community foundations vs generic DAF platforms:

  • F-39: Geographic grant map (local community visualization)
  • F-42/F-67: Grant catalog (curated local opportunities)
  • F-65: Event calendar & RSVP (community engagement)
  • F-64: Curated guidance content (philanthropic advisor role)
  • F-72: Community impact visibility (shared purpose)
  • F-66: In-portal messaging (personal relationship with staff)
  • F-62: Pledge tracking (CF-specific giving pattern)
