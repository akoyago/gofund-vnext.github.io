1. Authentication & Identity

1.1 Login (Email + Password)

  • Standard email/password authentication
  • Validates user has fund connections before allowing login
  • Manages timezone offset and IANA timezone claims on sign-in
  • Detects when 2FA is globally enabled and user hasn't set it up yet
  • Configurable sign-in instructions text and allow/disallow sign-ins toggle
  • Publishes UserSignInEvent on successful login
  • Account lockout support
